# 🚔 Smart Crime Analytics & Police Investigation System

A console-based **C++ Data Structures & Algorithms (DSA)** project that simulates crime case management, criminal record storage, investigation tracking, criminal network analysis, and crime analytics.

## 📌 Project Overview

This project demonstrates the practical application of various DSA concepts in a crime investigation environment. It provides a menu-driven system for managing crime cases, processing reports, analyzing criminal networks, and generating crime analytics.

## ✨ Features

* Crime Case Management
* Crime Report Processing
* Investigation History & Undo Operations
* Criminal Database Management
* Criminal Network Analysis
* Crime Analytics Dashboard
* Preloaded Dummy Data for Testing

## 🧠 DSA Concepts Implemented

* Bubble Sort
* Linear Search
* Stack
* Queue
* Binary Search Tree (BST)
* AVL Tree
* Hashing
* Graph
* Breadth First Search (BFS)
* Depth First Search (DFS)

## 🛠️ Tech Stack

* C++
* Object-Oriented Programming (OOP)
* Console-Based Application

## 📂 Project Modules

1. Crime Case Management
2. Sort Cases by Severity
3. Search Records
4. Crime Reports Queue
5. Investigation History
6. Criminal Database (BST)
7. Criminal Database (AVL)
8. Suspect Lookup (Hashing)
9. Criminal Network Analysis
10. Crime Analytics Dashboard

## ▶️ How to Run

### Compile

```bash
g++ -std=c++17 main.cpp -o main
```

### Run

```bash
./main
```
